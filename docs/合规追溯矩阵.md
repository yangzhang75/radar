# 合规追溯矩阵（Compliance Traceability Matrix）

> 型式认证（CAT 1）证据骨架。每条 = 一项标准要求 → 实现位置 → 验证证据。
> 各模块员工在落实标准条款时**追加/更新对应行**。编排者评审。状态：⬜待办 / 🔨实现中 / ✅已实现+验证。
> 原文 PDF：`桌面/雷达开发资料/标准资料/`、协议在 `通讯协议/`。

## 决策基线（已关闭）
- ✅ **送型式认证**，按最高合规标准。标准不再"省着用"，触及即逐条提取落实。
- ✅ 远程连接：蒲公英 X5 SD-WAN VPN，旁路+组播透明转发，软件按**同网段组播 + HALO 握手(01B1/01B2)**实现；MTU≤1400、容忍 1–3s 延迟/丢包、提供手动 IP 兜底。
