## 已解决的协议歧义（核对 SDK 头）
- `sampleEncoding` 取值：**SmpEncode_Amplitude=0、SmpEncode_Doppler=1**（值2/3保留）。来源 `SDK/include/NRPClient/NavRadarSpoke.h`。T0.3/T1.2 假设正确。
- 图像包前缀 `0100 0000 0020 0002` 中 `0x20=32` = `Spoke[32]` 数组容量，**非本包实际辐条数**。解析须按 `spokeLength_bytes` 步进遍历整个数据报（T1.2 已如此）——不可信任前缀计数。
